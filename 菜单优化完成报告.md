## 🔄 菜单-页签联动

### 1. 双向同步机制 ✅

#### 菜单 → 页签
```
点击菜单项
  ↓
router.push(item.path)
  ↓
路由变化
  ↓
TabsContainer 自动创建/激活页签
```

#### 页签 → 菜单
```
切换页签
  ↓
route.path 更新
  ↓
MainLayout 传递 route.path 到 AppSidebar
  ↓
AppSidebar 传递 currentPath 到 NavigationMenu
  ↓
NavigationMenu 计算 activeMenuKey 和 expandedKeys
  ↓
Menu 组件自动选中和展开
```

### 2. 智能父级展开 ✅

```typescript
// 递归查找所有父级路径
const expandedKeys = computed(() => {
  if (!props.currentPath) return []
  
  const findParentPaths = (items, targetPath, parentPath?) => {
    for (const item of items) {
      if (item.path === targetPath) {
        return parentPath ? [parentPath] : []
      }
      if (item.children) {
        const found = findParentPaths(item.children, targetPath, item.path)
        if (found) {
          return parentPath ? [parentPath, ...found] : found
        }
      }
    }
    return []
  }
  
  return findParentPaths(props.items, props.currentPath)
})
```

### 3. 自动选中当前项 ✅

```typescript
// 查找匹配当前路径的菜单项
const activeMenuKey = computed(() => {
  if (!props.currentPath) return null
  
  const findMenuItemByPath = (items, path) => {
    for (const item of items) {
      if (item.path === path) return item
      if (item.children) {
        const found = findMenuItemByPath(item.children, path)
        if (found) return found
      }
    }
    return null
  }

  const found = findMenuItemByPath(props.items, props.currentPath)
  return found?.path || null
})
```
