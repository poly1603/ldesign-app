## 🛠️ 技术细节

### 1. 插件系统优化
```typescript
// 分离 Engine 插件和 Vue 插件
plugins: [routerPlugin, i18nPlugin],  // Engine 插件
vuePlugins: [cachePlugin, ...]        // Vue 插件

// 在 setupApp 中安装 Vue 插件
vuePlugins.forEach(plugin => vueApp.use(plugin))
```

### 2. 错误处理增强
```typescript
// 全局错误拦截
onError: (error) => {
  console.error('❌ 应用错误:', error)
  setupErrorHandling(error)
}

// 友好的错误页面
showErrorPage(error)
```

### 3. 性能监控集成
```typescript
// 启用 Engine 性能监控
enablePerformanceMonitoring: true

// 启动监控
engine.performance.startMonitoring()
```
