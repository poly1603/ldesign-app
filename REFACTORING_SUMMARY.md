## 🔍 已知问题（可忽略）

### 1. 翻译键警告
**现象：** i18n 翻译键显示为原始 key（如 `home.title`）

**原因：** i18n 配置需要调整本地化文件

**影响：** 不影响功能，仅显示问题

**解决：** 后续补充翻译文件

### 2. 插件注册警告
**现象：** 某些 Vue 插件注册时的警告

**原因：** Engine 插件系统与 Vue 插件系统的兼容性

**影响：** 不影响核心功能运行

**解决：** 后续优化插件注册逻辑

### 3. 性能监控 API
**现象：** engine.performance 方法调用失败

**原因：** 需要完整的 engine 配置

**影响：** 已使用降级方案，功能正常

**解决：** 后续完善 engine 配置
